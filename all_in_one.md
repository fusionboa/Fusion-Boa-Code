# FusionBoa Language

FusionBoa is a **polyglot programming language** that compiles to multiple targets.

## Features

- **Multi-Target**: One file, many outputs
- **English-Like**: Readable by everyone
- **14+ Languages**: From Python to Rust to HTML

## Quick Start

Run: fusionboa build app.fusboa
Run: fusionboa run app.fusboa
